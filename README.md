# Vibecap API server

[vibecap](#) 애플리케이션과 통신하는 서버

## 기능
* 회원가입, 로그인
* 커뮤니티 (게시글, 댓글 작성, 좋아요, 알림)
* capture (사진, 사용자의 기분, 현재 상황과 관련된 플레이리스트 추천)

## 기술 스택
* application : `spring boot` `jpa`
* deployment : `aws ec2` `docker` `python`
* database : `mysql`
* etc : `google cloud` `youtube data`
